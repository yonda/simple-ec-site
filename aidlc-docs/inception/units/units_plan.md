# ユニット分割計画

## 目的

ユーザーストーリーを並行実装可能な作業ユニットに分割する。
各ユニットは境界づけられたコンテキスト（Bounded Context）として、特定のサブドメインまたはビジネスロールに沿った高い凝集性を持つ。
