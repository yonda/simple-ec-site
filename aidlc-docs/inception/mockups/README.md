# 書籍ECサイト - 画面モック

このフォルダには、シンプルな書籍ECサイトの画面モックが含まれています。HTML、CSS、JavaScriptのみで実装されたシンプルなプロトタイプです。

## ファイル構成

```
mockups/
├── README.md              # このファイル
├── index.html             # 商品一覧ページ（トップページ）
├── product-detail.html    # 商品詳細ページ
├── cart.html              # ショッピングカート
├── checkout.html          # 注文確認ページ
├── order-complete.html    # 注文完了ページ
├── login.html             # ログインページ
├── register.html          # 新規登録ページ
├── order-history.html     # 注文履歴一覧
├── order-detail.html      # 注文詳細ページ
├── styles.css             # 共通スタイルシート
├── data.js                # モックデータ（商品、注文履歴）
└── utils.js               # 共通ユーティリティ関数
```

## 使い方

1. **ブラウザでファイルを開く**
   - `index.html` をブラウザで開いてください
   - ローカルファイルとして動作します（サーバー不要）

2. **機能の確認**
   - 商品一覧の閲覧
   - 商品検索・フィルタリング
   - 商品詳細の表示
   - ショッピングカートへの追加
   - 注文フロー（注文確認→決済→完了）
   - ユーザー登録・ログイン
   - 注文履歴の確認

## 主要な機能

### 1. 商品カタログ（US-001 〜 US-004）
- 商品一覧表示（20冊のサンプル書籍）
- 商品詳細表示
- 検索機能（タイトル、著者名）
- フィルタリング（カテゴリ、価格帯、在庫状態）

### 2. ショッピングカート（US-005 〜 US-008）
- カートへの商品追加
- カート内容の表示
- 数量変更
- 商品削除
- LocalStorageでカートデータを保持

### 3. 注文・決済（US-009 〜 US-011）
- 注文確認画面
- 配送先情報入力
- 決済処理（モック、常に成功）
- 注文完了画面

### 4. ユーザー管理（US-012 〜 US-014）
- ユーザー登録（バリデーション付き）
- ログイン（モック認証）
- ログアウト
- LocalStorageでユーザー情報を保持

### 5. 注文履歴（US-015 〜 US-016）
- 注文履歴一覧（ログイン必須）
- 注文詳細表示
- サンプル注文データを表示

### 6. 在庫管理（US-017 〜 US-019）
- 購入時の在庫自動減算（モック）
- 在庫切れ商品の表示制御
- 在庫状態の表示（在庫数、残りわずか、在庫切れ）

## データ構造

### 商品データ（data.js）
```javascript
{
    id: 1,
    title: '書籍タイトル',
    author: '著者名',
    publisher: '出版社',
    isbn: 'ISBN番号',
    price: 3800,
    stock: 12,
    category: 'tech', // tech, novel, business
    description: '商品説明'
}
```

### カートデータ（LocalStorage）
```javascript
[
    {
        productId: 1,
        quantity: 2
    }
]
```

### ユーザーデータ（LocalStorage）
```javascript
{
    id: 1,
    email: 'test@example.com',
    name: '山田太郎'
}
```

## 制限事項

このモックは以下の制限があります：

- **データの永続化なし**: ページをリロードすると、LocalStorageのデータ以外はリセットされます
- **実際の決済処理なし**: 決済は常に成功として扱われます
- **サーバー通信なし**: すべての処理はクライアント側で完結します
- **セキュリティなし**: パスワードはハッシュ化されず、認証もモックです
- **在庫同期なし**: 複数ブラウザ間で在庫は同期されません
